# ReadyOn Time-Off Service

NestJS + SQLite implementation for the ReadyOn AI take-home exercise.

## Local Setup

```bash
npm install
npm run start:dev
```

The API defaults to `http://localhost:3000`.

## Useful Commands

```bash
npm run build
npm test
npm run test:e2e
npm run test:cov
npm run test:unit:cov
npm run lint
```

## Project Shape

- `src/` - NestJS application code
- `test/` - end-to-end tests and mock HCM helpers
- `docs/TRD.md` - technical requirements document
- `.github/workflows/ci.yml` - GitHub Actions CI
